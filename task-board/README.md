# Task Tracker App

 Create, edit, delete, and organize your tasks with ease.

# Features

- Create Tasks - Add new tasks with title and description
- Edit Tasks - Modify existing task details
- Delete Tasks- Remove tasks you no longer need
- Status Management - Track tasks as Todo, In Progress, or Done
- Filter Tasks - View tasks by status (All/Todo/Progress/Done)
- Persistent Storage - Tasks are saved automatically in your browser
- Responsive Design - Works on desktop, tablet, and 

## 🛠️ Tech Stack

- React 18- UI Library
- CSS3 - Styling
- LocalStorage API - Data persistence
- React Hooks - useState, useEffect
- Custom Hooks - useLocalStorage.jsx is for keep the data in local storage to access.
